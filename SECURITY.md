# Security Policy

## Supported versions

The latest released version receives security fixes.

## Reporting a vulnerability

Please report security issues **privately** via GitHub's
[Private Vulnerability Reporting](../../security/advisories/new). Do not open a
public issue for security problems.

We aim to acknowledge reports within 72 hours and to ship a fix or mitigation for
confirmed high-severity issues promptly. Reporters are credited unless you prefer to
remain anonymous.

## Scope

The Apace macOS application and its update mechanism (Sparkle, EdDSA-signed appcast).
