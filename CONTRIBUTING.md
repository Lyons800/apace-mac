# Contributing

Thanks for your interest. **Apace is source-available but proprietary** (PolyForm
Strict 1.0.0). The source is public for transparency and trust.

## We do not accept external code contributions

Because of the license, pull requests that add or modify code can't be accepted and
will be closed unmerged. Please don't invest time in code changes.

## What is welcome

- **Bug reports** and **feature requests** via [Issues](../../issues).
- **Security reports** — see [SECURITY.md](SECURITY.md). Please do **not** open a
  public issue for vulnerabilities.

## Filing a good bug report

Include your macOS version, the Apace version, clear reproduction steps, and any
relevant Console or crash logs.
